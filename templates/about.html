Advin: Search For Your Adventure

    Small Scale Search Engine With The Following Steps:

  • Scope: General search
  • Web Crawling: Web crawler to fetch web pages, using Scrapy, NLTK and BeautifulSoup
  • Indexing: Organize information gathered from web pages by parsing content, removing stop words, and creating an index that maps keywords to the documents where they appear
  • Ranking Algorithm: Determine the relevance of search results
  • Search Algorithm: Implemented a search algorithm to retrieve relevant search results based on user queries
  • User Interface: Used HTML CSS and JavaScript to create front end minimalist design

    • The Backend was created using Python
    The Frontend was created using HTML, CSS, and JavaScript
